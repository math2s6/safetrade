Documentation API

Toutes les requêtes nécessitent votre clé API en header.

Authorization: Bearer sk_live_xxxxxxxxxxxx...

Commandes

POST /api/b2b/orders Créer une commande 
curl -X POST https://safetrade.onrender.com/api/b2b/orders \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "external_order_id": "ORDER-123",
    "customer_email": "client@example.com",
    "customer_name": "Jean Dupont",
    "product_name": "Produit XYZ",
    "order_amount": 99.99
  }'
GET /api/b2b/orders Lister les commandes
PATCH /api/b2b/orders/:id/ship Marquer comme expédiée
PATCH /api/b2b/orders/:id/deliver Confirmer la livraison (ouvre l'unboxing)
GET /api/b2b/stats Statistiques de votre compte
